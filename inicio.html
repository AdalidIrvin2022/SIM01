DIV

La etiqueta DIV, permite insertar una DIVISIÓN en nuestra página, esta división facilita la estructuración de nuestro contenido, ya que permite organizar los diferentes elementos mediante CLASES e ID's.
El elemento DIV, puede contener a su vez tantos DIV, como sean necesarios. Un elemento equivalente es la etiqueta SECTION, que para efectos prácticos realiza la misma tarea.

JS

Funciones

Las funciones son fragmentos de código que nos permiten dividir las acciones en fragmentos auto-contenidos, de esta forma podemos crear fragmentos de código para solucionar partes de un problema y luego poder reutilizarlos.

Ejemplo: Dada una cadena de texto, que contiene un NOMBRE- APELLIDO P - APELLIDO M, crear una función que muestre el resultado en una lista formateada de la siguiente manera:

  • Nombre: nombre_introducido
  • Apellido Paterno: apellidoP_introducido
  • Apellido Materno: apellidoM_introducido
